# QTR AI v2

## О проекте

QTR AI v2 — это алгоритмическая торговая система, реализующая авторскую стратегию QTR Setup #1: Trend + Retest + Continuation.

Проект разрабатывается по принципу Architecture First.

## Главные правила проекта

1. Сначала спецификация — потом код.
2. Каждый модуль имеет одну ответственность.
3. Каждый модуль проходит отдельное тестирование.
4. Следующий модуль создается только после успешного теста предыдущего.
5. Архитектура не изменяется без обновления документации.

## Цель проекта

Создать надежного алгоритмического торгового бота, который:

- анализирует рынок;
- находит торговые сетапы;
- принимает решения;
- управляет рисками;
- проходит бэктест;
- работает на демо-счете;
- интегрируется с Telegram;
- после успешного тестирования может быть подготовлен к реальной торговле.
